╔══════════════════════════════════════════════════════════════════════════════╗
║  ⚠️ TROUBLESHOOTING                                                          ║
╚══════════════════════════════════════════════════════════════════════════════╝

PROBLEM: "Module not found" error when running train.py or app.py
SOLUTION: 
   pip install -r backend/requirements.txt

────────────────────────────────────────────────────────────────────────────────

PROBLEM: "Models not loaded" message in API
SOLUTION:
   1. Make sure you ran: python model/train.py
   2. Check backend/models/ folder has .joblib files
   3. Restart the API server

────────────────────────────────────────────────────────────────────────────────

PROBLEM: Frontend shows "Cannot connect to API"
SOLUTION:
   1. Make sure backend is running (python app.py)
   2. Check terminal shows "Uvicorn running"
   3. Visit http://localhost:8000/health to test
   4. Check API_BASE_URL in frontend/js/app.js is "http://localhost:8000"

────────────────────────────────────────────────────────────────────────────────

PROBLEM: Port 8000 already in use
SOLUTION:
   Option 1: Kill existing process using port 8000
   Option 2: Change port in backend/app.py:
      uvicorn.run("app:app", host="0.0.0.0", port=8001, reload=True)
      Then update frontend/js/app.js:
      const API_BASE_URL = 'http://localhost:8001';

────────────────────────────────────────────────────────────────────────────────

PROBLEM: Python not found
SOLUTION:
   1. Install Python 3.8 or higher from python.org
   2. During installation, check "Add Python to PATH"
   3. Restart terminal/PowerShell

────────────────────────────────────────────────────────────────────────────────

PROBLEM: Blank/white page in frontend
SOLUTION:
   1. Check browser console (F12) for errors
   2. Make sure you opened index.html in browser
   3. Try different browser (Chrome recommended)

────────────────────────────────────────────────────────────────────────────────

PROBLEM: Training takes too long or freezes
SOLUTION:
   1. Wait patiently (can take 2-3 minutes on slow machines)
   2. Check CPU/RAM usage
   3. Close other applications
   4. Reduce dataset size in train.py: generate_sample_dataset(n_samples=2000)

BACKEND FILES:
   
   app.py
      ↳ Main FastAPI server with all endpoints
      ↳ Handles HTTP requests from frontend
      ↳ Returns JSON responses
   
   model/train.py
      ↳ Generates synthetic dataset
      ↳ Trains XGBoost and Isolation Forest
      ↳ Saves trained models
      ↳ Run ONCE before starting server
   
   model/predict.py
      ↳ Loads saved models
      ↳ Preprocesses input data
      ↳ Makes predictions
      ↳ Validates input
   
   model/hybrid_logic.py
      ↳ Combines XGBoost + Isolation Forest
      ↳ Calculates weighted hybrid score
      ↳ Makes final fraud/genuine decision
   
   model/shap_explainer.py
      ↳ Generates SHAP values
      ↳ Explains feature contributions
      ↳ Creates human-readable summaries

FRONTEND FILES:
   
   index.html
      ↳ Web page structure
      ↳ Form fields for claim input
      ↳ Result display sections
      ↳ Admin dashboard modal
   
   css/style.css

      ↳ Animations
   
   js/app.js
      ↳ Form submission logic

      ↳ Result display
